
가상(화상) 키보드 기능을 구현한 프로그램

- SendInput() API를 사용해서 현재 입력포커스가 있는 컨트롤로 키 입력 메시지를 전달



개선해야 하는 사항

- 입력포커스가 있는 컨트롤에 대해 IME 입력 모드를 알아내는 기능 보완 필요
  . IME 프로그램의 한글/영문, CapsLock모드를 바로 알아내는 방법 필요
  . 웹브라우저에서 입력할 때 IME 입력 모드를 알 수 없음
  . 작업표시줄 검색창에서 입력할 때 IME 입력 모드를 알 수 없음
  . 파일탐색기 에서 경로를 입력할 때 IME 입력 모드를 알 수 없음
  => IME 프로그램과 직접 통신하며 입력모드를 알 수 있는 방법은 없음
